# StudyMate

[Live Site URL](https://your-live-site-url.com)

**StudyMate** is an interactive platform that helps students find study partners, collaborate on learning, and improve academic performance. It connects learners based on subjects, skills, and availability, making studying more efficient and enjoyable.

## Features

- **Top Study Partners**: Browse highly rated study partners with detailed profiles including subjects, experience level, and ratings.  
- **Secure Authentication**: Sign up and log in securely using email/password or Google authentication.  
- **Request & Connect**: Send requests to study partners and manage your pending connections.  
- **Dynamic Homepage**: Includes Banner, How It Works section, Top Partners, and Testimonials to provide an engaging user experience.  
- **Responsive Design**: Fully responsive UI for desktop, tablet, and mobile devices using modern React and Tailwind CSS.  
- **Real-Time Data**: Fetch partners and user requests dynamically from MongoDB with a secure Node.js and Firebase backend.

## How to Run Locally

1. Clone the repository:  
   ```bash
   git clone https://github.com/your-username/studymate.git
   ```  
2. Install dependencies for client and server:  
   ```bash
   cd client && npm install
   cd ../server && npm install
   ```  
3. Setup `.env` files with MongoDB URI and Firebase service key.  
4. Start the backend server:  
   ```bash
   npm run start
   ```  
5. Start the React frontend:  
   ```bash
   npm start
   ```
